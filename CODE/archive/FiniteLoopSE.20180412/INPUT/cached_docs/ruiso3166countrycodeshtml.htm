Коды стран – ИСО 3166

Что такое ИСО 3166?

ИСО 3166 является международным стандартом для представления названий стран и единиц их административно-территориального деления.

Целью разработки ИСО 3166 является определение всемирно признанных буквенных или цифровых кодов, которые мы можем использовать, когда ссылаемся на страны или их районы. Однако они не дают определений стран и их подразделений, данную информацию можно получить из источников ООН (Терминологический бюллетень названий стран или Кодов стран и регионов для статистических целей, которые ведет Статистический отдел ООН).

Применение кодов позволяет сохранять время и избегать ошибок. Вместо названия страны (которое будет изменяться в зависимости от используемого языка) мы можем использовать комбинацию букв и/или цифр, которые будут понятны во всем мире.

Например, все национальные почтовые организации по всему миру обмениваются международной почтой в контейнерах, маркированных соответствующими кодами стран. Интернет-системы доменных имен используют коды для определения доменных имен высшего уровня, таких как '.fr' для Франции, '.au' для Австралии. Кроме того, в машиночитаемых паспортах коды используются для определения национальности пользователя, а при банковских переводах позволяют определить место нахождения банка.

Каким образом можно получить доступ к ИСО 3166?

Коды, указанные в ИСО 3166 доступны на онлайн-платформе (OBP). Информация на OBP своевременно обновляется. На данные обновления возможно подписаться, нажав на соответствующий раздел в правом верхнем углу. 

Также у нас есть продукт, представляющий собой интернет-сборник кодов из 1, 2, 3 частей стандарта ИСО 3166 в трех различных форматах: .xml, .csv, и .xls для более легкого внедрения в вашу систему. Вы будете уведомлены о внесенных изменениях так что сможете загрузить последнюю версию. В данном случае вы можете быть уверены, что база данных содержит самую новую информацию от ИСО.

 

Что содержится в ИСО 3166?

ИСО 3166 имеет три части: коды единиц административно-территориального деления стран, коды территорий и ранее используемые коды (коды, которые использовались для описания стран ранее, но которые больше не используются).

Коды стран, которые могут быть представлены либо в виде двузначного кода (альфа-2), который рекомендуется в качестве кода общего назначения, или трехзначного кода (альфа-3), который может быть применен в тех случаях, когда применение латиницы неуместно.

Коды территорий представлены в виде альфа-2 кодов стран, доходя до трех символов. Например, ID-RI– это провинция Риау в Индонезии и NG-RI – это провинция Риверс в Индонезии. Названия и коды для подразделений обычно берутся из из соответствующих официальных национальных источников информации.

Ранее используемые - четырехзначные буквенные коды (альфа-4).

Узнайте больше о различных типах кодов в Глоссарии, разработанном для ИСО 3166.

Какая поддержка оказывается ИСО 3166?

ИСО 3166 регулярно пересматривается для того, чтобы соответствовать изменениям в названиях стран и их подразделениях. Данные изменения вносятся Агентством по техническому обеспечению (ИСО 3166/МА).

Агентство по техническому обеспечению включает следующие 14 организаций:

  • Французская ассоциация по стандартизации AFNOR (Франция);
  • Американский национальный институт стандартов ANSI (США);
  • Британский институт стандартов 
  • BSI (Великобритания);
  • Немецкий институт по стандартизации DIN (Германия);
  • Японский комитет промышленных стандартов (JISC);
  • Бюро стандартов Кении (KEBS);
  • Администрация по стандартизации Китая (SAC);
  • Шведский институт стандартов SIS (Швеция);
  • Международное агентство по атомной энергии (IAEA);
  • Международная организация гражданской авиации (ICAO);
  • Международный союз электросвязи (ITU);
  • Интернет-Корпорация по присвоению имен и номеров (ICANN);
  • Всемирный почтовый союз (UPU);
  • Европейская экономическая комиссия ООН (UNECE).

Как определяются элементы кодов стран?

Агентство по ведению ISO 3166 присваивает код независимой стране, определенной государством-членом Организации объединенных наций (ООН).

ООН уведомляет агентство по ведению ISO 3166, а также сообщает о присвоенном числовом коде. Затем ISO 3166/MA присваивает коды альфа-2 и альфа-3. Для других геополитических областей, которые являются частью суверенной страны и членом ООН, специальный запрос может быть подан в агентство по ведению ISO 3166.

Подлежит рассмотрению, что данные области не могут быть эксклавом, отделенным от основной территории страны, ни островом в территориальных водах материнской страны и должны быть физически отделены от основной территории.

 Такие просьбы должны поступать от компетентных должностных лиц национальных правительств стран, обладающих суверенитетом на данной области и ратифицированы или предложены в качестве альтернативы членом ИСО, обладающим суверенитетом в данной области.

How are country code elements assigned?

Following notification from the United Nations (UN), the ISO 3166 maintenance agency assigns alpha-2 and alpha-3 country code elements to new UN member states. The numeric country code is assigned by the UN.

As a general rule, the ISO 3166 maintenance agency does not assign official country code elements to other geopolitical areas or countries that are not member states of the UN.

However, the ISO 3166 maintenance agency may assign country codes to dependencies of countries that are member states of the UN. To be considered, some criteria, amongst others, include that the area be physically separated from its parent country and outside its territorial waters. Even if all criteria are met, the ISO 3166 maintenance agency may decide not to assign a code element, for example, due to the very limited number of official alpha-2 code elements available.
