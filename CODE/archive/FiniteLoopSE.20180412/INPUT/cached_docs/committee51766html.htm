ISO/TC 115

Pumps

 

About

  • Secretariat: AFNOR
    Secretary:
    Mme Sara Mélique
  • Chairperson (until end 2020):
    M Robert Dodane
  • ISO Technical Programme Manager :
    Mrs Kirsi Silander - van Hunen
    ISO Editorial Programme Manager :
    M Vincenzo Bazzucchi
  • Creation date: 1964

Scope

Standardization in the field of pumps; including:

  • dimensions and technical specifications,
  • test methods and conditions of acceptance for rotodynamic and positive displacement pumps for all liquids, including complete pumping installations,
  • installations and special applications, as well as rotodynamic mixers for waste water.
