ISO/TC 20

Aircraft and space vehicles

 

About

  • Secretariat: ANSI
    Secretary:
    Mr Christopher Carnahan
  • Chairperson (until end 2018):
    Mr James Rusty Rentsch
  • ISO Technical Programme Manager :
    Mr Andrew Dryden
    ISO Editorial Programme Manager :
    Mr Francesco Dadaglio
  • Creation date: 1947

Scope

Standardization of materials, components and equipment for construction and operation of aircraft and space vehicles as well as equipment used in the servicing and maintenance of these vehicles.
