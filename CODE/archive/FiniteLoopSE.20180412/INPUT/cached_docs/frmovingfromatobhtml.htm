Les normes ISO et le transport

Des sous-marins aux engins spatiaux, les normes ISO couvrent tous les aspects du transport, y compris les véhicules routiers, les navires, les aéronefs et les systèmes aérospatiaux.
Transporter les biens et les personnes en toute sécurité, et de manière efficace, est indispensable pour la solidité de l’économie comme pour la mobilité des personnes. Année après année, le nombre de trajets effectués est toujours plus important, de même que le rôle des normes ISO.
Qu’elles portent sur la conception, la construction et l’utilisation des véhicules, ou sur la manière dont les véhicules autonomes sont guidés, les normes ISO établissent une base commune pour renforcer la sécurité et limiter les chocs.

Ci-après, vous en saurez plus sur les normes le plus couramment utilisées dans le domaine des transports, sur les personnes qui les élaborent, et sur les normes en préparation.
